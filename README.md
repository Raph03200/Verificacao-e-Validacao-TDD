# Verificacao-e-Validacao-TDD
Exercício de Test Driven Development usando Jest

Teste TDD com uma função chamada "isPositive" que verifica se um número é positivo ou não, caso seja ele retornará true, caso seja 0 false, caso seja menor que 0 será lançado um erro e caso o usuário não envie um valor numérico o sistema lançará outro erro

# Para rodar o teste:

1. Baixe os módulos do node
```
npm install
```

2. Rode os testes Jest
```
npm test
```
